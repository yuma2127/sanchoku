# 農業と情報 課題：バックエンド機能付き産直市Webサイト

完全無農薬・無化学肥料・有機栽培にこだわった新鮮野菜の魅力を伝えるフロントエンドデザインと、Firebase Firestore をバックエンド（データベース）として活用したリアルタイムメッセージ投稿機能を統合したWebアプリケーションです。

## 1. 開発システム概要
*   **テーマ**: 「気合」をコンセプトにした無農薬・有機栽培野菜の産直市サイト
*   **フロントエンド**: HTML5, CSS3 (レスポンシブデザイン対応、黄緑ベース)
*   **バックエンド (BaaS)**: Firebase Cloud Firestore (v10 SDK / CDN経由)
*   **主なバックエンド機能**: 
    *   `addDoc` を用いた非同期メッセージ（口コミ・感想）のリアルタイム書き込み機能
    *   `getDocs` および `orderBy` を用いた、投稿日時順（新しい順）の自動メッセージ一覧表示

## 2. 実装におけるセキュリティ・安全面への配慮
*   **XSS（クロスサイトスクリプティング）対策**: 
    ユーザーから入力された名前（ニックネーム）やコメント本文を表示する際、`innerHTML` は一切使用せず、すべて `textContent` を経由して描画しています。これにより、悪意のあるスクリプトが混入・実行されるリスクを完全に排除しています。
*   **時刻の整合性確保**: 
    投稿時間の記録にはクライアント側の時計ではなく、Firebaseの `serverTimestamp()` を利用し、サーバー側で厳密にタイムスタンプを付与しています。

## 3. セットアップ・動作確認手順
1. 本レポジトリをクローンまたはダウンロードします。
2. `index.html` 内の `const firebaseConfig = { ... };` 部分に、ご自身のFirebaseプロジェクトのAPIキー等の接続構成情報を貼り付けます。
3. Firebase Firestore のセキュリティルールを、書き込み・読み込みが許可された状態（テストモード等）に設定します。
4. `index.html` をブラウザで直接開くか、ローカルサーバーを起動してアクセスします。
